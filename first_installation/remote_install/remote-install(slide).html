1. making ’Install USB’ control computer

1.1. build iso-image version of install OS

  • $ nix build .#nixosConfigurations.[install-OS-name].config.system.build.isoImage
  • $ ls -lah

1.2. copy install OS to the usb

  • $ lsblk (confirm [device-name-of-usb] and ’mount state’)
  • $ sudo fdisk dev[device-name-of-usb] >> m >> d (delete a partition) >> p (partition table) >> g (gpt) >> q (quit)
  • $ sudo dd if=[real-path-of-install-OS] of=/dev/[device-name-of-usb] bs=4M status=progress

1.3. eject the usb

  • $ lsblk (confirm ’mount state’)
  • $ umount [device-name-of-mounted-usb]
  • Eject the usb from computer

2. build nixos-configuration for target computer control computer

2.1. build target OS image

  • $ nix build .#nixosConfigurations.[target-OS-name].config.system.build.toplevel
  • ls -lah

3. target computer setting target computer

3.1. hardware setting

  • internet line connection, install USB insert
  • bios setup for usb booting (sometimes it needs to ’disable’ [Secure Boot])

3.2. format partitions (Gparted)

  • ’Gparted’
  • delete all partitions
  • making ’BOOT’ partition preceding 1MiB, size: 512MiB, Primary Partitions, no Partition name, fat32, Label: BOOT
  • making ’root’ partition preceding 0MiB, size: all the amount of remain, Primary Partitions, no Partition name, ext4, Label: root
  • Apply
  • Manage flags: set ’BOOT’ partitions as ’esp’ & ’boot’

3.3. internet connection confirm & get IP address

  • open ’terminal’
  • $ ifconfig
  • $ ping google.com

3.4. mount ’root’ & ’BOOT’ partition

  • $ sudo su
  • # mount /dev/disk/by-label/root /mnt
  • # mkdir /mnt/boot
  • # mount /dev/disk/by-label/BOOT /mnt/boot

4. nix copy control computer

4.1. ssh connection test

  • $ ssh root@[target-computer-IP]
  • {ssh}$ exit

4.2. nix copy to target computer

  • $ nix copy [real-path-of-target-OS] –to “ssh-ng://root@[target-computer-IP]?remote-store=/mnt” –no-check-sigs

5. nixos-install target computer

5.1. nixos-install

  • confirm the result of ’nix copy’ >> $ ls mnt/nix/store[real-path-of-target-OS]
  • # nixos-install –system nix/store[real-path-of-target-OS]
  • # reboot
  • Eject ’install USB’

6. ssh-key copy (if it needs to have same authority) control computer

6.1. ~./ssh copy from control computer to target computer

  • scp -r ~/.ssh [user-name]@[target-computer-IP]:/home/[user-name]/.ssh

7. apply home-manager target computer

7.1. git clone ’my-nixos’

7.2. apply home-manager

  • $ nix flake update
  • $ nix develop
  • $ home-manager switch –flake .#[home-configuration-name]
